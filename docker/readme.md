# Docker Orchestrator


## What build.sh does ?

 - Builds rest-api

 - Makes Java Docker Image out of rest-api

 - Builds phonegap , web-ui and preapres final ui build with native mobile apps

 - Makes nginx Docker image out of UI Build

  -  rest-api (phanimaridu/openams-rest-api:0.0.1)
